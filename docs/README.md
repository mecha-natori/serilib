# serilib ドキュメント

serilibのドキュメントのソースを置いています。
これらは`main`へのプッシュ・PRマージごとにビルドされ[ここ](https://mecha-natori.github.io/serilib)で公開される他、v0.1.1以降は配布物の`share/docs`配下に同梱されます。

## 構造

| ディレクトリ名 | 内容                                            | 公開パス   |
| :------------: | :---------------------------------------------- | :--------- |
|     `root`     | トップページ。ポータルの役割を担う。            | `/`        |
|    `manual`    | 基本的な使い方をまとめた説明書。mdBook製。      | `/manual`  |
|   `apidocs`    | 関数・定数などの詳細なリファレンス。Doxygen製。 | `/apidocs` |
