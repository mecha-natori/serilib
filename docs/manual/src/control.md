# 操作

このモジュールは`seri/control.h`として公開されている。主に操作データの変換などを集約している。

## スティックデータ

左右スティックの値をまとめて扱う型として`sticks_t`を実装している。これは各スティックの上下軸・左右軸をそれぞれ符号付き16bit値として保持し、更にデッドゾーンを百分率(整数)で保持している。上下軸は上が、左右軸は右が正である。

## 正規化スティックデータ

スティックデータをそれぞれ正規化した物を扱う型として`sticks_nor_t`を実装している。これは`normalize_sticks`関数に`sticks_t`を渡すことで得られる。`normalize_sticks`関数は各スティックの各軸に対して-1.0〜0.0〜1.0の範囲への正規化の他、デッドゾーンの処理も行う。そのため、`sticks_nor_t`にはデッドゾーン情報は含まれず、各軸のデータのみを保持している。
