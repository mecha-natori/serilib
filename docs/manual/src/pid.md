# PID制御

このモジュールは`seri/pid.h`として公開されている。主にPIDデータの処理などを集約している。

なお、足回りにおいては自動で使われるため、特にここの関数を呼び出す必要はない。

## PIDパラメーター

PIDパラメーターを保持する型として`pid_params_t`を実装している。これはPゲイン、Iゲイン、Dゲインを`SL_FP_T`型で格納したものである。

## PIDデータ

PID制御で用いるデータをまとめて保持する型として`pid_data_t`を実装している。これは、`pid_params_t`、前回処理時の差分、前回処理時点の積算差分、現在の実際の値を保持している。
この内、現在の実際の値である`actual_value`は処理前に手動で更新する必要がある。

## PID処理

PIDデータと目標値を元に実際の出力値を計算する関数として`process_pid`関数を実装している。これは`pid_data_t`へのポインタと`SL_FP_T`型の目標値を受け取り、`pid_data_t`を更新した上で計算された出力値を`SL_FP_T`型で返す。
