# 数学補助

このモジュールは`seri/math.h`として公開されている。主に標準ライブラリにはない数学関数などを集約している。

## 座標型

直交座標系を表す型として`cart_t`が実装されている。これはX・Y座標を`SL_FP_T`型でそれぞれ保持しているだけの型である。X座標は右が、Y座標は上が正である。また、極座標系を表す型として`polar_t`が実装されている。動径rと角度θを`SL_FP_T`型でそれぞれ保持している。θは右を0とし、反時計回りに増加していく。

それぞれ`create_cart`・`create_polar`関数で作成可能である。また、`cart2pol`・`pol2cart`関数で相互変換が可能である。

### ベクトル演算

`cart_t`を二次元ベクトル型とみなし、ベクトル演算を行う関数をいくつか実装している。`dot`関数はその名の通りベクトルのドット、つまりは内積(スカラー積とも)を求める関数である。2つのベクトルの内積を`SL_FP_T`型で返す。また`vec_scale`関数はベクトルをスカラー倍する関数である。ベクトル、`SL_FP_T`型のスケール値の順で渡すとスケールされたベクトルが返ってくる。

## 逆線形補間

区間`[a,b]`において、値`x`がどの割合の位置であるかを算出する逆線形補間を行うマクロとして`invlerp`を実装している。区間の始点、終点、値の順に引数に渡すことで結果が返ってくる。

## クランプ

区間`[a,b]`において、値`x`が範囲内であれば`x`、そうでなければ`a`・`b`の内近い方を返すマクロとして`clamp`を実装している。値、区間の始点、終点の順に引数に渡すことでクランプした結果が返ってくる。

## 近似等価比較

`==`による等価比較が出来ない浮動小数点数において近似等価比較を行う関数として`approx`・`approxf`・`approxl`関数を実装している。2値の差がその型のεより小さい場合に等価とみなすことで実装される。
