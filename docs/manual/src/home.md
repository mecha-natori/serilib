# serilib マニュアル

このマニュアルでは、serilibの各モジュールについて仕様例を交えつつ解説する。

## 浮動小数点数の扱いについて

本ライブラリにおいて、全ての浮動小数点数は`SL_FP_T`と言うマクロ型で扱われている。
これは、ビルド時にライブラリ内で使う浮動小数点数型を決められるようにするためのものである。

昨今の32bitマイコンは単精度FPUを積んでいる場合がある。
その際、ライブラリ内部における数値計算を少しでも高速化するため、あえて`float`を使う必要がある。
しかし、マイコンによっては倍精度FPUを積んでいることもある。その場合、より精度の高い`double`を使いたい所である。
それ故、ビルドプリセットから型を指定できる必要があったのだ。

なお、OS環境向けプリセットにおいては、性能が十分であろうという判断から`double`としている。
